YXY's Schedule

发表于 2022-04-10

作息

1.晚上尽量保证睡8小时
2.累的时候及时休息
3.感觉中午睡半个小时到一个小时更舒适

吃饭

1.早饭感觉不太想吃，一般起的太晚就不吃了；
2.午饭和晚饭，尽量可以吃点好吃的，心情会比较好；
3.夜宵一定要吃！！！不要犹豫，一定要吃，不吃直接自闭；
4.如果看手机不舒服的话，就不要看了，会很影响吃饭体验；

看的东西

很多东西都 没来得及/没心情 仔细看
1.公众号
2.各种群
3.书
4.朋友圈
5.B站视频
6.聊天

上午

1.起床尽快进入状态；
2.一般都有课，上课（软工、毛概）适当选择要做的事；
3.上午状态感觉都不太行，尽量保持比较清醒吧；
4.如果效率不太行的话可以处理些生活事务；

中午

1.吃完饭回到宿舍尽快睡，睡够觉最舒服！（一个小时比较够）
2.可以订个下午的水果；

下午

1.感觉下午效率一般还是在线的，保持心态和精神状态良好，劳逸结合；
2.如果效率不太行的话可以处理些生活事务；
3.未完待续……
4.晚饭可以早点吃，这样夜宵吃的更香😋

晚上

1.晚上一般效率最高了，保持状态，高效工作，劳逸结合；
2.如果累就先睡一觉，睡觉可香了；
3.闹钟：23:00看书，23:30洗脚，23:59洗漱睡觉；
4.感觉晚上时间有点难安排，想/需要做的事：视频、看书、吃夜宵、写博客、玩LOL、写作业/写代码；
5.写数学/写代码属于是白天的事没做完，拖到晚上做了；
6.玩LOL不能太晚，容易兴奋而难以入睡，推荐时间为22:00-23:00（容易超时，看情况调整）；
7.晚上实在睡不着就下来做些事情（超算、博客、看书、听音乐），可以继续吃夜宵，这时候其实挺安静挺舒服的；

高效工作，劳逸结合

发表于 2022-04-05 更新于 2022-04-10

高效工作

1.戴耳机学习/干活好像效率更高，因为心情会更好，而且更容易专注
2.番茄工作法，每25/45分钟休息一下，活动活动，做点别的

劳逸结合

1.如何深度放松（还没好好看）；
2.适合自己的放松方式：睡觉、听音乐+唱歌、玩LOL、看书？
3.专注工作 -> 适当放松奖励，放松的时候不要有负罪感
4.累的时候可以的话就休息，不要在情绪状态不好时硬撑着做事情，效率会低而且心情会差；（比如上完课之后可以躺会儿，或是睡觉，听音乐）
5.生活有很多繁琐的事情，每天也不一定会过的充实美满，情绪低落时好好放松，让低落期尽快过去就好；（观点来自xyh和钜导）

New Beginning

发表于 2022-03-28 更新于 2022-04-10

新的开始

1.好好睡觉：00:30-08:30 + 适当小憩
2.好好吃饭：早饭（同夜宵/吃清汤面+抄手）、午饭（随意）、晚饭（随意）、夜宵（泡面/面包+酱+水果+奶）、水果（多样）
3.好好学习：要尽量认真地对待每一门课（课件、课本、作业）
4.好好听课：能线上就线上，听课的时候不干别的事，专心听（当然是值得听的课才好好听课）
5.多多运动：约人游泳，踢球？健身！（还有50米、引体向上、阳光长跑）
6.少少摸鱼：以能休闲且有意义的方式代替，如：看书、看新闻评论、听音乐
7.劳逸结合：该学习/干活时专注，适当放松（番茄工作法）
8.自我提升：看书，中宏经，睡前消息，写博客，记录爱情

旧的开始

1.好好睡觉：00:30-08:30（睡不着的话适当延后，但要保证8个小时） + 累了就睡，不要犹豫
2.好好吃饭：早饭（同夜宵 / 吃清汤面+抄手 / 点粥喝）、午饭（要吃好吃的！）、晚饭（要吃好吃的！）、夜宵（拌面 / 泡面 / 面包+酱+水果+奶）、水果（多样，可以试试果切）
3.好好学习：要尽量认真地对待每一门课（课件、课本、作业），投入适当的时间，进行适当的学习
4.好好听课：能线上就线上，听课的时候不干别的事，专心听（当然是值得听的课才好好听课）
5.多多运动：约人游泳，踢球？健身！（还有50米、引体向上、阳光长跑）
6.少少摸鱼：尽量以能休闲且有意义的方式代替，如：看书、看新闻评论、听音乐
7.劳逸结合：该学习/干活时专注，适当放松（番茄工作法）
8.自我提升：看书，中宏经，睡前消息，写博客，记录爱情
9.限制游戏：完成作业或任务时再奖励性的玩，不能毫无限制（三局)
